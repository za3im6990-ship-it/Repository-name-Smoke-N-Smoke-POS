# Smoke N Smoke POS - Android App

تطبيق Android POS باسم Smoke N Smoke POS.

## Settings
- App name: Smoke N Smoke POS
- Currency: USD
- Language: Arabic + English
- Temporary logo included

## Features
- POS sale screen
- Add product, cost, sale price, quantity
- Auto total and profit
- Save sales locally with SQLite
- Daily report
- Dark / Neon design

## Build APK easiest way
This project includes GitHub Actions workflow:
`.github/workflows/build-apk.yml`

Upload this project to a GitHub repository, then:
Actions > Build APK > Run workflow

The APK will appear in Artifacts as:
Smoke-N-Smoke-POS-debug-apk
